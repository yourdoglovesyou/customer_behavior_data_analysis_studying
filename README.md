Customer Shopping Behavior Analysis
End-to-End Data Analytics Project (Python · SQL · Power BI · Gamma)

Overview
This project demonstrates a complete data analytics workflow, from raw data ingestion to business insights delivery. The goal is to analyze customer shopping behavior using Python and SQL, visualize insights with Power BI, and communicate findings through a professional report and presentation.

The project showcases practical skills in:

Data cleaning and exploratory data analysis (EDA)

SQL querying on relational databases

Dashboard design and storytelling

Executive-ready reporting and presentation

Dataset
Source: Customer Shopping Behavior Dataset (CSV)

Format: Tabular transactional data

Key Features (examples):

Customer demographics

Purchase frequency and spending

Product categories

Transaction-level details

Tools & Technologies
Python: pandas, numpy, matplotlib, seaborn

SQL: PostgreSQL / MySQL / SQL Server

Database: Relational SQL database

Visualization: Power BI

Reporting: PDF report

Presentation: Gamma (AI-powered PPT creation)

Project Workflow & Steps
1. Data Loading & Cleaning (Python)
Loaded raw CSV data into Python

Handled missing values and data type issues

Created derived features for analysis

Performed exploratory data analysis (EDA)

2. Database & SQL Analysis
Imported cleaned data into a SQL database

Wrote SQL queries to:

Aggregate customer spending

Analyze purchase patterns

Segment customers by behavior

Validated results against Python EDA

3. Dashboard Development (Power BI)
Connected Power BI to SQL Server

Built an interactive dashboard with:

Key KPIs (total sales, average spend, frequency)

Customer segmentation views

Category and trend analysis

Designed with clarity and stakeholder usability in mind

4. Reporting & Presentation
Summarized insights in a structured analytics report

Created a professional presentation using Gamma

Focused on business impact, not just technical results

Dashboard
The Power BI dashboard provides:

High-level performance metrics

Drill-down analysis by customer segment

Visual trends for decision-making support

File: customer_behavior_dashboard.pbix

Results & Key Insights
Identified high-value customer segments

Discovered purchasing trends across categories

Highlighted opportunities for targeted marketing

Demonstrated how data-driven insights can support business strategy
